# fpush

fpush是即时消息推送服务程序. <br/>
旨在做一个类似于极光推送，小米推送之类的Java程序开源实现。基于Netty + protobuf
<br/>

## 技术栈
1. JDK1.8 <br/>
2. Netty-4.1.31.Final <br/>
3. protobuf-java 3.6.1 <br/>

## 运行
eclipse/IDEA里
Step1 右键run as--java application-- FpushServer.java <br/>

Step2 右键run as--java application-- FpushClient.java <br/>
